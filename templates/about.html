what's boa?


boa is a web-based automated reverse engineering platform that helps extrapolate original and readable Python source from a compiled and packed executables. It undergoes the following when an executable sample is processed through its pipeline:


  • Executable Unpacking - parse out resources from Py2Exe and PyInstaller binaries.
  • Bytecode Decompilation/Patching - decompile and patch (if necessary) Python source from unpacked bytecode.
  • Static Analysis - scan relevant source files for detrimental security issues.
  • Report Generation - generate a user-friendly report on executable.

At the current moment, boa's functionality is still being implemented. PE executables are currently only supported for unpacking and decompilation, and there is still very limited support for virtualization-based deobfucation.


what can I use it to reverse?


  • closed black-box applications
  • malware samples
  • ctf and wargame challenges

how much does it cost?


boa is offered as open-sourced freeware. However, a paid version with much more functionality is being implemented formore professional reverse engineering. See pricing for more information.
